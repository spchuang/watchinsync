Hackathon Project

This is a project made by Sam Chuang and Justin Wei during the 2012 Facebook hackathon at UCLA. We made a few changes to the original app that was built within 24 hours. It is built on Nodejs, Express, and Socketio, utilizing youtube api and tokbox api. The app is deployed with forever module.
